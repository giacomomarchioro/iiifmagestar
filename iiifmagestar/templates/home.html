IIIF Image API 3.0 Server

This server implements the IIIF Image API 3.0 specification with enhanced features.

API Endpoints

Image Info (JSON-LD):
GET /iiif/3/{identifier}/info.json
Image Request:
GET /iiif/3/{identifier}/{region}/{size}/{rotation}/{quality}.{format}

New Features in 3.0

Square Region:
/iiif/3/example/square/300,300/0/default.jpg
Automatically extracts the largest centered square from the image
Arbitrary Rotation:
/iiif/3/example/full/full/23.5/default.jpg
Rotate by any angle, not just 90Â° increments
Best Fit Sizing:
/iiif/3/example/full/!800,600/0/default.jpg
Fit within specified dimensions while preserving aspect ratio
Upscaling Allowed:
/iiif/3/example/full/^1200,/0/default.jpg
Allow upscaling beyond original dimensions

Standard Examples

Full image as JPEG:
/iiif/3/example/full/max/0/default.jpg
Crop region:
/iiif/3/example/100,100,500,500/max/0/default.jpg
Percentage crop:
/iiif/3/example/pct:10,10,80,80/max/0/default.jpg
Resize to 300px wide:
/iiif/3/example/full/300,/0/default.jpg
Percentage resize:
/iiif/3/example/full/pct:50/0/default.jpg
Rotate and mirror:
/iiif/3/example/full/full/!90/default.jpg
Grayscale PNG:
/iiif/3/example/full/max/0/gray.png

Supported Formats

Standard: jpg, png, webp
Extended: tif, tiff

Supported Quality Levels

default, color: Full color
gray: Grayscale conversion
bitonal: Black and white (1-bit)
